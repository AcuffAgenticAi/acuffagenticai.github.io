# 🤖 Agentic Marketing Attribution AI

This repository hosts an **Automated Marketing Attribution Agent** that analyzes sales data, calculates profitability (ROAS), and updates a live dashboard.

## 🚀 How It Works
1. **Data Input**: Marketing data is stored in `sales/marketing_data.csv`.
2. **AI Agent**: The Python script `sales/run_agent.py` processes the data using `pandas` and `seaborn`.
3. **Automation**: A GitHub Action (`.github/workflows/run_agent.yml`) triggers every time the CSV is updated.
4. **Live Visualization**: The agent generates `roas_report.png`, which is automatically displayed on the [Live Site](https://acuffagenticai.github.io/).

## 📊 Project Structure
* `index.html`: The frontend dashboard.
* `sales/`:
    * `run_agent.py`: The core logic for attribution and ROAS calculation.
    * `marketing_data.csv`: The source of truth for all marketing transactions.
    * `roas_report.png`: The latest visual insight generated by the agent.
* `.github/workflows/`: Contains the automation pipeline.

## 🛠️ Local Setup
To run the agent locally:
1. Install dependencies:  
   `pip install -r sales/requirements.txt`
2. Run the agent:  
   `python sales/run_agent.py`
